
# План: Переименование "Библиотека ролей" → "Библиотека промптов"

## Обзор

Пункт меню "Библиотека ролей" открывает редактор системных промптов для ИИ-ролей. Название не соответствует содержимому — это библиотека **промптов** (системных инструкций), а не самих ролей. Переименуем для ясности.

## Изменения

| Местоположение | Было | Станет |
|----------------|------|--------|
| Главное меню | Библиотека ролей | Библиотека промптов |
| Заголовок страницы | Библиотека ИИ-ролей | Библиотека промптов |

## Затрагиваемые файлы

| Файл | Изменения |
|------|-----------|
| `src/contexts/LanguageContext.tsx` | Обновление ключей переводов |

## Детали изменений

### Обновление локализации

```typescript
// src/contexts/LanguageContext.tsx

// Меню навигации
'nav.roleLibrary': { ru: 'Библиотека промптов', en: 'Prompt Library' },

// Заголовок страницы
'roleLibrary.title': { ru: 'Библиотека промптов', en: 'Prompt Library' },
```

## Примечание

Все остальные ключи локализации (`roleLibrary.new`, `roleLibrary.edit` и т.д.) оставляем без изменений, так как они относятся к конкретным промптам внутри библиотеки и уже корректно описывают действия с ними.
