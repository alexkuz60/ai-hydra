
# План: Добавление Гидропедии

## Обзор
Создание нового раздела "Гидропедия" — интерактивного руководства пользователя по работе с AI-Hydra. Раздел будет содержать структурированную документацию в формате markdown с описанием концепций, функций и лучших практик работы с платформой.

## Изменения

### 1. Навигация (AppSidebar)

**Файл:** `src/components/layout/AppSidebar.tsx`

Добавить новый пункт меню "Гидропедия" с иконкой `BookOpen`:
- Расположение: после "Рейтинг моделей"
- Путь: `/hydrapedia`
- Локализация: `nav.hydrapedia`

### 2. Локализация

**Файл:** `src/contexts/LanguageContext.tsx`

Добавить переводы:
- `nav.hydrapedia`: "Гидропедия" / "Hydrapedia"
- Заголовки и подразделы страницы

### 3. Страница Hydrapedia

**Новый файл:** `src/pages/Hydrapedia.tsx`

Страница с markdown-документацией:
- Боковая навигация по разделам
- Рендеринг markdown контента
- Стилизация в духе проекта (HydraCard)

### 4. Контент Гидропедии

**Новый файл:** `src/content/hydrapedia.ts`

Структурированный markdown-контент:

```text
СОДЕРЖАНИЕ ГИДРОПЕДИИ
=====================

1. ВВЕДЕНИЕ
   - Что такое AI-Hydra
   - Ключевые концепции
   - Архитектура многоголовой системы

2. НАЧАЛО РАБОТЫ
   - Регистрация и вход
   - Настройка профиля
   - API-ключи

3. ПАНЕЛЬ ЭКСПЕРТОВ
   - Выбор моделей
   - Одновременный запрос к нескольким "головам"
   - Таймауты и управление ответами
   - D-Chat консультант

4. РОЛИ АГЕНТОВ
   - Assistant (Эксперт)
   - Critic (Критик)
   - Arbiter (Арбитр)
   - Consultant (Консультант)
   - Moderator, Advisor, Archivist, Analyst, Webhunter

5. БИБЛИОТЕКА ПРОМПТОВ
   - Создание промптов
   - Назначение ролей
   - Шаринг промптов

6. ИНСТРУМЕНТЫ
   - Промпт-инструменты
   - HTTP API инструменты
   - Тестирование инструментов

7. РЕДАКТОР ПОТОКОВ МЫСЛИ
   - Создание диаграмм
   - Типы узлов
   - Экспорт в различные форматы

8. РЕЙТИНГ МОДЕЛЕЙ
   - Система оценок
   - Статистика использования

9. СОВЕТЫ И ЛУЧШИЕ ПРАКТИКИ
   - Эффективные промпты
   - Работа с несколькими моделями
   - Когда использовать какую роль
```

### 5. Маршрутизация

Добавить маршрут:

## Технические детали

### Структура страницы Hydrapedia

```text
+------------------+------------------------+
|   Sections Nav   |    Content Area        |
|                  |                        |
| > Введение       |  ## Что такое AI-Hydra |
|   Начало работы  |                        |
|   Панель экспертов|  AI-Hydra — это...    |
|   Роли агентов   |                        |
|   Библиотека     |  ### Ключевые концепции|
|   Инструменты    |                        |
|   Редактор потоков|  - Мультиагентность   |
|   Рейтинг        |  - Синергия моделей    |
|   Советы         |  - Роли и специализация|
+------------------+------------------------+
```

### Технологии
- `react-markdown` для рендеринга (уже в проекте)
- `MarkdownRenderer` компонент (уже есть в проекте)
- `HydraCard` для стилизации секций
- Навигация через состояние React (без дополнительных роутов)

### Пример структуры контента

```typescript
interface HydrapediaSection {
  id: string;
  titleKey: string; // ключ локализации
  content: {
    ru: string;
    en: string;
  };
}
```

Это позволит динамически переключать язык документации синхронно с интерфейсом.
