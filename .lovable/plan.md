
# Расширение левых навигационных панелей под десктоп

Проект ориентирован на Desktop-First. Сейчас несколько навигаторов имеют слишком узкие дефолтные размеры, не использующие доступное пространство экрана. Нужно привести все панели к более широким десктопным значениям.

## Текущее состояние и целевые значения

| Страница | Текущий размер (%) | Новый размер (%) |
|---|---|---|
| Expert Panel | 20 | 25 |
| Flow Editor | 15 | 18 |
| Model Ratings | 20 | 25 |
| Staff Roles | 35 | 35 (без изменений) |
| Behavioral Patterns | 35 | 35 (без изменений) |
| Tasks | 40 | 40 (без изменений) |
| Role Library | 40 | 40 (без изменений) |
| Tools Library | 40 | 40 (без изменений) |

Страницы Tasks, RoleLibrary, ToolsLibrary, StaffRoles и BehavioralPatterns уже имеют достаточно широкие панели (35-40%). Основные изменения коснутся трёх панелей с заниженными значениями.

Также будет увеличен глобальный дефолт в хуке `useNavigatorResize` с 25% до 30%, чтобы любые будущие навигаторы по умолчанию были шире.

## Технические детали

Изменения в файлах:

1. **src/hooks/useNavigatorResize.ts** -- `DEFAULT_MAX_SIZE`: 25 -> 30
2. **src/pages/ExpertPanel.tsx** -- `defaultMaxSize`: 20 -> 25
3. **src/pages/FlowEditor.tsx** -- `defaultMaxSize`: 15 -> 18
4. **src/pages/ModelRatings.tsx** -- `defaultMaxSize`: 20 -> 25

Пользователи, у которых уже сохранены размеры в localStorage, продолжат использовать свои настройки -- изменения затронут только первый визит или после очистки хранилища.
