

# Сохранение концепции Подиума в файл

## Задача

Создать файл `PODIUM_CONCEPT.md` в корне проекта с полным описанием стратегической концепции Подиума ИИ-моделей -- как видение пользователя, так и анализ с предложениями.

## Содержимое файла

Файл будет включать:

1. **Общая концепция** -- Подиум как аналог Штата специалистов, но для ИИ-моделей; помощь в оптимальном выборе моделей для задач
2. **Портфолио** -- досье модели: участие в задачах, роли, оценки, галлюцинации, критика
3. **Конкурс интеллект-красоты** -- тестирование кандидатов на роли (Эксперт, Консультант, Критик, Арбитр); выявление слабых сторон; помощь в создании кастом-промптов
4. **Рейтинги** -- сводная статистика по всем моделям и задачам; фильтры, кросс-таблицы, сводные итоги
5. **Анализ и предложения** -- расширение schema (model_statistics, model_contests, contest_entries), фазы реализации, открытые вопросы

## Примечание по ошибкам сборки

Текущие ошибки -- это временный rate limit (429) на стороне инфраструктуры Cloudflare/R2, не связанный с кодом. Они разрешатся автоматически при следующей сборке.
